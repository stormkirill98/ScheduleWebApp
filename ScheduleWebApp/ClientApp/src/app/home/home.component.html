Вы вошли в приложении для создания и просмотра учебного расписания:

  • Директор - может редактировать расписание; создавать новые группы, преподавателей, дисциплины; настраивать права для пользователей
  • Учитель - может просматривать расписание по определенному преподавателю
  • Студент - может просматривать расписание по определенной группе

Если в navbar у вас нет ни одной вышеупомянутой вкладки, то обрадитесь к администратору для настройки прав
